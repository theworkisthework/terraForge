# terraForge
Plotter control and gcode generator

Claude Sonnet 4.6 Vibe Coded Electron App to control a terrapen pen plotter.

It currently allows importing an SVGs, PDF & Gcode.

PDFs and SVGs can be moved, scaled and roated, converting to and saving as, gcode to local machine and/or uploading directly to SD Card on the controlller. 

It also supports previwing and plotting goode files from SD. 

It has jog controls, file browser, console and machine config.
